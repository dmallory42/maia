## Maia v0.1.0

Initial public release of `maia/framework`.

### Included

- API-first PHP framework with app kernel, dependency injection, routing, middleware, exceptions, logging, and HTTP testing helpers
- ORM support for models, relationships, migrations, schema building, joins, grouping, `having`, and `upsert`
- Auth and security helpers for JWT, API keys, CORS, rate limiting, validation, and security headers
- CLI scaffolding for apps, controllers, services, models, middleware, requests, tests, and migrations
- SQLite bootstrap helpers and generic response caching middleware

### Release readiness

- CI coverage for PHP 8.2, 8.3, and 8.4
- Scaffold smoke test for a freshly generated app
- GPL-3.0-or-later licensing
- User and contributor documentation split into `README.md` and `CONTRIBUTING.md`
